\# Variáveis e Tipos de Dados na Lógica de Programação



\## 1. Introdução



Na lógica de programação, variáveis e tipos de dados são conceitos fundamentais para a construção de algoritmos. Eles permitem armazenar informações, manipulá-las e utilizá-las na resolução de problemas computacionais.



Antes de aprender uma linguagem específica, é essencial compreender esses conceitos de forma abstrata.



---



\## 2. Variáveis



\### 2.1 Conceito



Uma variável é um espaço reservado na memória do computador que recebe um nome e armazena um valor. Esse valor pode ser alterado durante a execução do algoritmo.



Podemos pensar em uma variável como uma "caixa identificada", onde guardamos um dado para usar depois.



\### 2.2 Estrutura básica



Uma variável possui:



\- Nome (identificador)

\- Tipo de dado

\- Valor



\### 2.3 Exemplo em pseudocódigo



```



idade ← 20

nome ← "Ana"

altura ← 1.70



```



Nesse exemplo:



\- `idade` armazena um número inteiro

\- `nome` armazena um texto

\- `altura` armazena um número real



---



\## 3. Regras para nomear variáveis



Em geral, as variáveis:



\- Devem começar com letra

\- Não podem conter espaços

\- Não podem usar palavras reservadas da linguagem

\- Podem conter números após a primeira letra

\- Podem usar o caractere underscore (\_)



Exemplos válidos:



```



idade

mediaFinal

quantidade\_alunos



```



Exemplos inválidos:



```



2idade

media final



```



---



\## 4. Tipos de Dados



Os tipos de dados definem que tipo de valor uma variável pode armazenar. Eles também determinam quais operações podem ser realizadas com esse valor.



\### 4.1 Inteiro



Armazena números sem casas decimais.



Exemplos:



```



idade ← 25

quantidade ← 100



```



\### 4.2 Real (ou Decimal)



Armazena números com casas decimais.



Exemplos:



```



preco ← 19.90

altura ← 1.75



```



\### 4.3 Caractere



Armazena apenas um único caractere.



Exemplo:



```



letra ← 'A'



```



\### 4.4 Cadeia de Caracteres (String)



Armazena um conjunto de caracteres, ou seja, textos.



Exemplos:



```



nome ← "Carlos"

mensagem ← "Bom dia!"



```



\### 4.5 Booleano (Lógico)



Armazena apenas dois valores possíveis:



\- verdadeiro

\- falso



Exemplo:



```



aprovado ← verdadeiro



```



---



\## 5. Tipagem



A tipagem refere-se à forma como os tipos de dados são tratados.



\### 5.1 Tipagem Estática



O tipo da variável deve ser declarado explicitamente antes do uso e não pode ser alterado.



Exemplo:



```



inteiro idade ← 18



```



\### 5.2 Tipagem Dinâmica



O tipo é definido automaticamente de acordo com o valor atribuído.



Exemplo:



```



idade = 18



```



---



\## 6. Importância na Lógica de Programação



Variáveis e tipos de dados são importantes porque:



\- Permitem armazenar informações temporárias

\- Organizam os dados dentro do algoritmo

\- Evitam erros de operação

\- Tornam o código mais claro e estruturado



Sem variáveis, não seria possível criar programas que interagem com dados ou realizem cálculos.



---



\## 7. Conclusão



As variáveis são responsáveis por armazenar dados durante a execução de um algoritmo. Já os tipos de dados determinam a natureza dessas informações e como elas podem ser manipuladas.



Dominar esses conceitos é essencial para avançar no estudo da lógica de programação e no aprendizado de qualquer linguagem de programação.
